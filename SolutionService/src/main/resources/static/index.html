Welcome to Equalities Solution Service!

Service needs params to work:
 - firstValue : Required
- secondValue : Required
- leftBorder, rightBorder : Optional

Possible user requests:

1) /solve/url - GET Request
Solves equality using input parameters received from GET request in Json format. Otherwise, service returns exception.

2) /solve/json - POST Request
Solves equality using input parameters received from POST request in Json format. Otherwise, service returns exception.

3) /solve/bulk - POST Request
Solves equalities using list of input parameters received from POST request in Json format. If there are incorrect input params service returns list of calculated roots from correct ones.
